- Rule B1 (Identity Verification): Before discussing any account details or taking actions on the customer's account, the agent MUST verify the customer's identity by completing necessary verification and security checks. Failing to perform or complete identity verification is a direct violation.

- Rule B2 (Escalation Rights): If a customer requests to escalate the call, speak to a supervisor, or speak to a manager, the agent MUST offer to connect them to a supervisor. Refusing an escalation request is a direct violation.

- Rule B3 (Call Closure and Transfers): When concluding a call or transferring a customer, the agent MUST confirm the outcome, set clear expectations and timelines for next steps, and provide a case reference number if an issue was raised. If transferring, the agent must advise the customer of the hold time and take preliminary actions beforehand. Failing to provide a reference number, timeline, or leaving issues unresolved without follow-up is a violation.
